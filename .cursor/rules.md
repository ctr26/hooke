# Cursor Rules

## Context

This is the hub repo for the Virtual Cells ecosystem. It is docs-only — no source code.

## Standards

- Conventional commits: `feat:`, `fix:`, `docs:`, `test:`, `chore:`
- Read STANDARDS.md and GUIDELINES.md before making changes
- Each doc ≤8k tokens
- No duplicate content across docs
- Single source of truth — link, don't copy

## Skills

Skills are available at `.claude/skills/`. Use them:
- `what` — before coding, understand requirements
- `how` — before acting, understand workflow
- `why` — before deciding, understand rationale
- `where` — before placing, understand context

## Don't

- Don't overwrite existing docs without explicit request
- Don't commit secrets or credentials
- Don't create docs that exceed 8k tokens
- Don't duplicate content — link to the source
